Bem-vindo!

Este site é um sistema de gerenciamento de Pokémons para a mansão do Raito.

Como utilizar o sistema

  1. Selecione um clã no menu lateral para visualizar os Pokémons disponíveis.
  2. Adicione os Pokémons que deseja utilizar clicando no botão +.
  3. Insira sua senha de treinador cadastrada e confirme para registrar o uso.
  4. Para devolver, acesse o histórico ou a lista de Pokémons em uso e use sua senha.
  5. Você pode devolver todos os Pokémons ou apenas alguns específicos.
  6. Gerencie treinadores (adicionar/remover) no botão "Treinadores" no cabeçalho (requer senha admin).
